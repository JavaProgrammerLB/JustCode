## Git
## 每次看代码时把代码更新到最新
```
1.列出当前配置的所有远程仓库

git remote -v
2.增加上游

git remote add upstream https://github.com/spring-projects/spring-framework.git
3.测试上游增加结果

git remote -v
4.选择要更新的分支

git checkout master
5.从上游更新代码

git fetch upstream
git merge upstream/master
6.同步到自己的github仓库

git push
```

- IO 要考虑到的点：同步、异步、是否阻塞
